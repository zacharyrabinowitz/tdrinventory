WINDOWS SETUP

1) Open VS Code in this folder.

2) Create virtual environment:
   py -m venv .venv

3) Activate it:
   .venv\Scripts\activate

4) Install dependencies:
   pip install -r requirements.txt

5) Run:
   py app.py

6) Open:
   http://127.0.0.1:5000
